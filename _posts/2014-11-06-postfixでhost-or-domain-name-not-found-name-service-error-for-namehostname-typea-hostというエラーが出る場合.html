CentOSのPostfixで
Host or domain name not found. Name service error for name=hostname type=A: Host not found
などというエラーが出て上手くいかない時

設定が合っている場合でもOS側でsendmailが選択されている場合があります。

sudo alternatives --config mtaコマンドでpostfixを選択してみてください。
